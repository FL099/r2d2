# r2d2
Versuch mit einem raspberryPi einen funktionierenden(zumindest theoretischen) R2D2 zu bauen  

## Überlegungen

* Sprache: Python
* Falls Web: Flask
* Sensoren:
  * Wasser
  * Entfernung
  * Mikrophon
  * Kamera
  * Infrarot (o-ä - Fernbedienung)
  * usw
